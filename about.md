## 你好，我是咸蛋超人迪卡

也可以叫我 **KaiDecker**。

我目前就读于北京林业大学统计学专业，预计 2028 年毕业。专业训练让我习惯先看数据、假设和评价标准；而在课程与项目里，我逐渐把兴趣移向了 **Agent、后端、机器学习和测试开发**。

<p class="secondary-copy" lang="en">I am a Statistics undergraduate at Beijing Forestry University, expected to graduate in 2028. My current interests sit around agent engineering, backend systems, machine learning, software testing, and data analysis.</p>

我未来希望跨专业攻读计算机相关硕士。虽然目前还是本科阶段，我已经完成了一轮计算机 408 的系统学习，并继续通过项目和课程把这些知识从“看过”变成“用过”。

## 学习路径 / Learning path

### 统计学

统计学不是我打算丢掉的背景。它让我更在意数据从哪里来、结论能否解释、指标有没有误导，以及一个看起来不错的结果是否真的可靠。

### 计算机基础

我已经系统学习过 408 的主要内容：

- 数据结构
- 计算机组成原理
- 操作系统
- 计算机网络

也完成过 **CS61A、CS61B 和 CS50**，通过课程练习接触编程抽象、数据结构、算法、C、Python、Java 等内容。

### 工程与项目

目前更想深入的方向包括：

- Agent 工程、评测与可观测性
- 后端开发与系统基础
- 机器学习实验与工程化
- 测试开发、自动化与质量保障
- 统计建模和真实数据分析

我写过 Python、Java 和 C++。这里不放熟练度百分比，因为语言只是工具，真正能说明问题的还是具体项目、代码和复盘。
